1、集合
（1）定义：由一个或多个确定的元素所构成的整体。
（2）特点：元素类型不一定相同；无序。

2、列表【线性列表】
（1）定义：是一种数据项构成的有限序列，即按照一定的线性顺序，排列而成的数据项的集合。
（2）特点：有序；长度可变。
（3）存储方式：连续存储和不连续存储（如：链表）。
（4）表现形式：
1）常见类型：数组、链表
2）特殊类型：栈、队列

3、数组
（1）数组是列表的实现方式之一。
（2）不同语言中差异：
1）C++ 和 Java 中，数组中的元素类型必须保持一致，而 Python 中则可以不同。
2）Python 中的数组叫做 list，具有更多的高级功能。
（3）宏观上区分列表和数组：索引。
（4）存储方式：连续存储（且每个元素占用相同大小的内存）。
